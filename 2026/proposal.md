# TECH x ROCK Festival 2026 企画書

## コンセプト

テック系エンジニアを対象にした参加型ライブイベント。
固定メンバーのいないぼっちプレイヤーでも気軽に参加できるよう、個人単位で募集してバンドを組む形式。

- 技術ミートアップのノリでワイワイ音楽を楽しむ
- なるべく多くのプレイヤーが楽しむ。みんな主役
- プレイヤーも観客として楽しむ。家族や友人も一緒に

## 2025年からの変更点

- **個人枠のみの募集に一本化**
  - よりコミュニティ感を重視した形にするため、バンドメンバー同士の交流がしやすい個人枠に絞る
- **バンドマッチングをAI化**
  - 申込データをもとにAIが複数のバンド案を生成し、オーガナイザーが確認・調整する形に

## システム構成（方針）

従来のGoogleフォーム＋手作業JSON運用から、Webアプリベースの仕組みに移行する。

- **参加申込**: 専用Webアプリで受付。入力データはデータベースに格納
- **バンドマッチング**: 締切後、ハナ（AI）がDBのデータを参照して自動的にバンド案を生成。jyoshiseが確認・調整して確定
- **Webサイト反映**: 確定後、ハナがbands.json等を更新してWebサイトに反映

技術スタックの詳細は別途検討。

## 参加費の目安（初参加の方へ）

TRFへの参加にかかる費用のおおまかな目安です。

| 費目 | 金額の目安 | 備考 |
|---|---|---|
| 参加費 | 5,000円 | ハコ代・運営費の分担 |
| 打ち上げ | 3,000〜5,000円程度 | 任意 |
| スタジオ練習代 | 1,000〜2,000円/回 | バンド人数で割り勘。回数はバンド次第 |
| 器材・消耗品 | 数百〜数千円 | 弦・スティックなど。既に持っていれば不要 |

スタジオ練習は多くの場合2〜4回程度になることが多いです。
合計すると **1〜2万円程度** を目安にしておくと安心です。

## 運営体制

過去の参加者を中心に、役割ごとに担当を分担する形を予定しています。

| 役割 | 担当 | 主な作業 |
|---|---|---|
| オーガナイザー | jyoshise | 企画全体の意思決定、最終承認、対外発信 |
| 会場担当（ボランティア） | 過去参加者から選定 | 会場との交渉・契約・事務手続き、各バンドのセット図取りまとめ・提出、当日の会場スタッフ対応 |
| データ・コンテンツ担当 | AI（ハナ） | 申込データ集計、バンドマッチング、Webサイト更新、告知文・連絡文案作成 |

会場担当は過去参加者の中から募る予定。詳細は企画確定後に声がけします。

## スケジュール（案）

| フェーズ | 時期 | 主な内容 |
|---|---|---|
| Phase 1: 準備 | 〜2026/6月末 | 企画確定、申込システム構築、Slack準備 |
| Phase 2: 募集 | 2026/7月上旬 | 参加申込受付開始 |
| Phase 3: マッチング | 2026/7月中旬 | バンド仮組み → 参加者間で調整 → 確定 |
| Phase 4: 会場・日程確定 | 2026/7〜8月 | バンド数・参加者数をもとに会場選定、日程をSlackで参加者と相談して確定 |
| Phase 5: 開催準備 | 2026/8〜10月 | 各バンドでスタジオ練習、タイムテーブル確定、Webサイト更新 |
| Phase 6: 本番 | 2026/11〜12月 | ライブ本番（2days予定） |

**募集開始まで約2週間。** 準備を急ぎで進める必要があります。

## バンド編成の考え方

- ドラムの人数がそのままバンド数の上限になる
- ドラムの得意ジャンルを起点にバンドの方向性を仮設定し、他パートを当てはめる
- 掛け持ちは原則なし（なるべく多くの人に機会を与えたい）
- ドラム不足が予想されるため、募集時はドラムを特にアピール
