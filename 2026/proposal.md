# TECH x ROCK Festival 2026 企画書

## コンセプト

テック系エンジニアを対象にした参加型ライブイベント。
固定メンバーのいないぼっちプレイヤーでも気軽に参加できるよう、個人単位で募集してバンドを組む形式。

## 2025年からの変更点

- **固定バンド枠を廃止**し、個人枠のみの募集に一本化
  - 固定バンドでの参加者はSlack等でのコミュニティ内交流が少なく、イベントの一体感を損ねる傾向があったため
- **バンドマッチングをAI化**
  - 申込データをもとにハナ（AI）が複数のバンド案を生成し、ジョシセが確認・調整する形に

## 運営方針

2025年まで実質一人で運営していたが、規模拡大に伴い以下をAI（ハナ）に委託：

| タスク | 担当 |
|---|---|
| 参加申込フォーム設計 | ハナ（叩き台） + ジョシセ確認 |
| 申込データ集計・分析 | ハナ |
| バンド仮組み（マッチング） | ハナ |
| バンドプロフィール作成・bands.json反映 | ハナ |
| タイムテーブル作成・events.json反映 | ハナ |
| Webサイト更新（MR作成） | ハナ |
| 告知文・リマインダー文案 | ハナ |
| 最終確認・意思決定・対外連絡 | ジョシセ |

## スケジュール（案）

| フェーズ | 時期 | 主な内容 |
|---|---|---|
| Phase 1: 準備 | 〜2026/2月 | 企画確定、フォーム設計、connpass/Slack準備 |
| Phase 2: 募集 | 2026/3月 | 参加申込受付（個人枠のみ） |
| Phase 3: マッチング | 2026/4月 | バンド仮組み → 参加者間で調整 → 確定 |
| Phase 4: 開催準備 | 2026/5〜8月 | タイムテーブル確定、Webサイト更新、リマインダー |
| Phase 5: 本番 | 2026/秋 | ライブ本番 |

## バンド編成の考え方

- ドラムの人数がそのままバンド数の上限になる
- ドラムの得意ジャンルを起点にバンドの方向性を決め、他パートを当てはめる
- 掛け持ちは原則なし（なるべく多くの人に機会を与える）
- ドラム不足が予想されるため、募集時はドラムを特に呼びかける

## 参加費

- プレイヤー参加費: 5,000円程度を想定（スタジオ代・ハコ代の分担）
- 詳細は会場費・バンド数確定後に算出
